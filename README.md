# compendium-des-tables-aleatoires
Un module pour FoundryVTT regroupant des tables aléatoires crées par la communauté pour le système Star Wars FFG. Ce compendium vise à réunir les tables aléatoires proposés par différents contributeurs. Actuellement, il comprend :



PERSONNAGES

Générateur de noms

Liste des espèces

Clientèle de bar

Liste de personnages d'aucun secours

Particularités de droid

Particularités de personnalité



OBJECTS

Bibelots



VAISSEAU

Maintenance de vaisseau

Pannes d'hyperdrive

Particularités de vaisseau

Cargo
